# Project Statement: Personal Command-Line Budget Tracker

## 1. Problem Statement
The difficulty individuals face is maintaining a reliable, accessible, and simple record of income and expenses without incurring costs or navigating complex commercial financial software. There is a need for a lightweight, easily auditable tool for persistent personal finance tracking.

## 2. Scope of the Project
The project's scope is strictly limited to the core functionality of recording financial transactions and calculating the net balance on a single user's local machine. 


## 3. Target Users
The primary target users are individuals seeking a simple, non-subscription based tool for daily personal finance tracking. It is also suitable for students or educational users who require a basic, functional finance simulator for learning purposes.

## 4. High-Level Features
1.  **Data Input & Validation:** Simple, interactive command-line interface for clear input of transaction details (amount, category).
2.  **Data Persistence:** Reliable storage and retrieval of all records using a local JSON file.
3.  **Financial Reporting:** Calculation and display of the Current Balance (net worth) in real-time.

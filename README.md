# Budget-Tracker-Project
This Python-based Budget Tracker addresses the challenge of managing personal finances effectively. It allows users to input income and categorize expenditures (eg: bills, groceries and other expenses) offering real-time financial summaries. The project applies foundational programming concepts to ensure data persistence and userfriendly management
# Personal Command Line Budget Tracker
This project is a Python-based command-line application designed to help users track their personal finances. It allows users to record daily income and expenses, view a summary of their net balance, and categorize spending for better financial management. All transaction data is saved persistently using a JSON file.
## Features
The Budget Tracker provides the following core capabilities:
* **Transaction Logging:** Allows users to input new income or expense transactions, recording the amount, date, and category.
* **Persistent Storage:** All data is automatically saved to a local
* **budget_data.json** file, ensuring data persists across application sessions.
* **Balance Reporting:** Displays a real-time summary of the current net balance (Total Income - Total Expenses).
* **Categorization:** Enables tracking of spending by category (e.g., Food, Salary, Rent).
* **Command Line Interface (CLI):** Provides a clean, menu-driven interface for easy interaction.
## Technologies Used
* **Language:** Python 3.13.5 
* **Core Modules:** `json`  and `datetime`.
